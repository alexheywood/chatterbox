
# Chatterbox

A single page application (SPA) chat room web app that allows multiple clients to talk to one another built with React and Socket.io.



## Run Locally

Clone the project

```bash
  git clone https://github.com/alexheywood/chatterbox.git
```

Client

```bash
  cd chatroom
  npm install
  npm run dev
```
----
Server

```bash
  cd server
  npm install
  npm start
```
